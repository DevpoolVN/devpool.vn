# Astrolabe documentation

Start here if you're new. Each page below is short and self-contained.

| Page                                       | Read this when…                                                              |
| ------------------------------------------ | ---------------------------------------------------------------------------- |
| [Getting started](./getting-started.md)    | You want to clone, configure, and ship your first post.                      |
| [Writing posts](./writing-posts.md)        | You're about to write a post and want to know the frontmatter & conventions. |
| [Customizing](./customizing.md)            | You want to change colors, fonts, nav, layout, the site title.               |
| [Architecture](./architecture.md)          | You want to understand how the pieces fit before you start editing.          |
| [Features](./features.md)                  | You want to know what each built-in feature does (and how to remove it).     |
| [Skills](./skills.md)                      | You want to use the `.agents/skills/` agent skills with Claude / Cursor / Codex / etc. |

If you're in a hurry: clone, `npm install`, `npm run dev`, edit `src/config/site.ts`, write a post in `src/content/blog/`. The rest is detail.
